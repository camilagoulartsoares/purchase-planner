# Meu Closet dos Sonhos

Lista de compras e desejos pessoal para organizar roupas, calçados, bolsas e acessórios — com preços, promoções, orçamento e status.

## Stack

- React + TypeScript + Vite
- Tailwind CSS v4
- Lucide Icons
- Persistência em `localStorage`

## Desenvolvimento

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Funcionalidades

- Cadastro, edição e exclusão de peças
- Upload de imagem ou URL
- Filtros, busca, ordenação e abas por status
- Visualização em cards ou lista
- Cálculo automático de descontos e totais
- Orçamento mensal com barra de progresso
- Dados salvos no navegador
