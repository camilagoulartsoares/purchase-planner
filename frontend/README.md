# Frontend

Interface do projeto pessoal Purchase Planner.

Deploy: https://purchase-planner.vercel.app

## Tecnologias

- React
- TypeScript
- Vite
- Tailwind CSS
- React Router
- Axios
- Lucide React

## O Que Foi Feito

- Rotas protegidas.
- Dashboard e listagem principal.
- Filtros e busca.
- Cards com galeria e ações.
- Páginas de marcas e detalhes.
- Modal de criação e edição.
- Skeleton loading.
